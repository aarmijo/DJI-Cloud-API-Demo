**Direction:** up

**Method:** drc_camera_photo_info_push

**Data:**

| result                              | Capturing result         | int                              |                                                                                                                                                                       | Non-zero represents an error |
| status                              | Status                   | enum_string | {"fail":"Failed","in_progress":"In progress","ok":"Finished"}                                    |                              |
| progress                            | Progress                 | struct                           |                                                                                                                                                                       |                              |
